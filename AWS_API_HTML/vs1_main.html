AWS 서비스 아키텍처 개요

AWS Lambda
이벤트 기반의 고가용성 서버리스 컴퓨팅 플랫폼입니다. 별도의 인프라 자원 예약 및 서버 운영 관리 작업 없이, 호출 빈도와 실행 시간에 비례한 정밀한 비용 최적화 아키텍처를 구현합니다.
AWS RDS
복잡한 인프라 관리 부담을 해소하는 엔터프라이즈급 풀 매니지드 관계형 데이터베이스 서비스입니다. 스토리지 자동 확장, 자동 패치 및 Multi-AZ 가용성 복제를 완벽하게 지원합니다.
AWS Secrets Manager
데이터베이스 자격 증명 및 API 토큰 등 핵심 기밀 정보를 런타임 시점에 안전하게 제공하는 중앙 집중형 보안 솔루션입니다. 자동 암호 회전 정책을 기본으로 지원합니다.
