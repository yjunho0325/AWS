AWS Secrets Manager

핵심 특징

  • 자동 기밀 회전(Rotation): 교체 주기를 지정해 두면 사람이 개입하지 않아도 데이터베이스 패스워드를 알아서 변경하고 최신화합니다.
  • 강력한 데이터 암호화: AWS KMS(Key Management Service) 마스터 키와 연동되어 데이터가 물리 디스크에 보관되거나 API 통신으로 전달될 때 무조건 암호화 처리됩니다.
  • 런타임 동적 로딩: 백엔드 로직이 필요할 때마다 동적으로 원격 API를 조회하여 메모리 단에서만 인증 처리를 마친 후 소거하므로 유출 가능성을 원천 차단합니다.
